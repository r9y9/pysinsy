pysinsy
===========

A python wrapper for `Sinsy <https://github.com/r9y9/sinsy>`_.

https://github.com/r9y9/pysinsy


Installation
------------

The latest release is availabe on pypi. You can install it by:

.. code::

   pip install pysinsy
